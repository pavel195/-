# ГЭК Тренажёр

Локальный сайт для подготовки к ГЭК: тестовые вопросы, устные вопросы и отдельный раздел с практическими заданиями.

## Как запустить

### Вариант 1. Просто открыть файл
Открой `index.html` в браузере.

### Вариант 2. Через локальный сервер
```bash
cd gec-trainer-site
python3 -m http.server 8000
```

Затем открой:

```text
http://localhost:8000
```

На Windows можно использовать:

```bash
python -m http.server 8000
```

## Что внутри

```text
index.html        главная страница
css/styles.css    стили сайта
js/app.js         логика тренажёра
data/bank.js      общий банк вопросов и практики
```

## Разделы сайта

- **Экзамен** — билет как на ГЭК: 12 тестов, 3 устных, 1 практика.
- **Тесты** — тренировка тестовых вопросов по теме.
- **Устные** — вопросы с эталонными ответами.
- **Практика** — отдельный банк практических заданий.
- **Банк** — статистика и поиск по всем вопросам.

## Как добавлять свои вопросы

Открой `data/bank.js` и добавь объект в нужный массив:

```js
{
  "id": "t999",
  "topic": "Python",
  "question": "Что делает метод pop()?",
  "options": ["Удаляет по индексу", "Удаляет по значению", "Очищает список", "Сортирует"],
  "answer": 0,
  "explanation": "list.pop(index) удаляет и возвращает элемент по индексу.",
  "difficulty": "base"
}
```

Сложность может быть:

```text
base, medium, hard
```

## Как использовать с ChatGPT

В разделе **Практика** нажми **«Скопировать промпт»**, вставь его в чат и добавь своё решение. После этого можно попросить проверить как комиссия.
